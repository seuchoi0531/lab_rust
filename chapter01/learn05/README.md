## 01-05
숫자 타입을 다룰 때는 주의해야 함
타입이 같아야만 연산이 가능하기 때문에, `u32`와 `u8`을 더할 수 없음

그럴 때 `as` 키워드를 사용해서 숫자 타입을 변환할 수 있음
```
let a = 13u8; // u8
let b = 7u32; // u32
let c = a as u32 + b; // u32 = u32 + u32
```

`u8`을 `u32`로 바꾸는 것도 가능하고, `u32`를 `u8`로 바꾸는 것도 가능함

`bool` 타입은 숫자 타입으로 변환하면 `true`는 `1`, `false`는 `0`으로 변환함
