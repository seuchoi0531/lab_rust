===
조건문은 다른 언어에서 많이 본 `if`, `else if`, `else`를 사용해서 작성
조건문의 조건 부분은 괄호를 사용하지 않고 작성
조건문의 실행 부분은 중괄호를 사용해서 작성

```
let x = 42;
if x < 42 {
  // 42보다 작을 때의 실행 부분
} else if x == 42 {
  // 42와 같을 때의 실행 부분
} else {
  // 42보다 클 때의 실행 부분
}
```

다른 언어에서 많이 본 관계 연산자와 논리 연산자를 그대로 사용함
- `==`, `!=`, `<`, `>`, `<=`, `>=`, `!`, `||`, `&&`
