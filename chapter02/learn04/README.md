## 02-04
반복문은 `for` 키워드로도 작성가능
`for`은 얼마나 반복하는 범위를 명시해야 함
반복하는 범위는 `x in n..m`와 같은 형태로, `in` 키워드를 사용하여 작성
- `x`를 `n`부터 `m` 전까지 반복하는 범위

```
for x in 0..5 {
  println!("{}", x);
}
// 0
// 1
// 2
// 3
// 4
```

`n..=m`으로 범위를 설정하면 `m`까지의 범위

```
for x in 0..=5 {
  println!("{}", x);
}
// 0
// 1
// 2
// 3
// 4
// 5
```
